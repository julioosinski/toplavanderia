🚀 CONFIGURAÇÃO RÁPIDA - ESP32 AutoConfig v3.0 (Top Lavanderia)
================================================================

✅ NÃO PRECISA EDITAR NENHUMA LINHA DO CÓDIGO!

O ESP32 v3.0 é plug-and-play. Basta fazer upload do arquivo
ESP32_AutoConfig_v3.ino e configurar pelo celular.

================================================================

📋 PASSO A PASSO:

1️⃣ UPLOAD DO CÓDIGO
   - Abra o arquivo ESP32_AutoConfig_v3.ino no Arduino IDE
   - Selecione a placa: "ESP32 Dev Module"
   - Faça upload (mesmo código para TODOS os ESP32s)

2️⃣ PRIMEIRO BOOT
   - O ESP32 pisca o LED lentamente
   - Ele cria uma rede WiFi: TopLav_XXXXXX
   - E ativa o Bluetooth (BLE) com nome TopLav_XXXXXX

3️⃣ CONFIGURAR WiFi (escolha uma opção):

   Opção A - Pelo Navegador (mais fácil):
   - Conecte ao WiFi "TopLav_XXXXXX" (sem senha)
   - Abra o navegador em http://192.168.4.1
   - Preencha: SSID, Senha do WiFi, Laundry ID
   - Clique "Salvar e Conectar"

   Opção B - Por Bluetooth (BLE):
   - Use um app BLE (ex: nRF Connect)
   - Conecte ao dispositivo "TopLav_XXXXXX"
   - Envie JSON na característica WiFi Config:
     {"ssid":"SUA_REDE","pass":"SUA_SENHA","laundry_id":"UUID"}

4️⃣ O ESP32 REINICIA E SE REGISTRA AUTOMATICAMENTE
   - Conecta ao WiFi configurado
   - Envia heartbeat com auto_register=true
   - Aparece no painel admin como "Pendente"

5️⃣ APROVAR NO PAINEL ADMIN
   - Vá em: Admin → Configurações
   - Na seção "ESP32s Pendentes de Aprovação"
   - Associe o ESP32 a uma máquina
   - Clique "Aprovar"

================================================================

🔑 ONDE ENCONTRAR O LAUNDRY ID:
   - Painel Admin → Configurações → QR Code de Configuração
   - Copie o UUID ou escaneie o QR Code

================================================================

🔄 RECONFIGURAR WiFi (se a rede mudar):
   - Segure o botão BOOT do ESP32 por 5 segundos
   - O LED pisca rápido e o ESP32 faz reset de fábrica
   - Repita o passo 3

================================================================

❌ PROBLEMAS COMUNS:

1. "Não encontro a rede TopLav_XXXXXX"
   → O ESP32 pode estar em modo normal (já configurado)
   → Faça reset de fábrica (botão BOOT por 5 segundos)

2. "WiFi conectou mas não aparece no painel"
   → Verifique se o Laundry ID está correto
   → Aguarde 30 segundos para o heartbeat

3. "ESP32 reinicia em loop"
   → Verifique se a rede WiFi é 2.4GHz (5GHz não funciona)

================================================================

📌 TopLavanderia_v4_BLE.ino — compilação (Arduino IDE + pacote ESP32 3.x)

• Erro "Sketch too big" / "text section exceeds available space"
  → Ferramentas → Esquema de partições → Huge APP (3MB No OTA) ou No OTA (2MB APP)
     (o padrão ~1,3 MB de APP não cabe WiFi + BLE Bluedroid + HTTP + ArduinoJson).

• Erro JsonDocument(4096) com ArduinoJson 7
  → Use: JsonDocument doc;  (sem número no construtor).

• Upload falha: "The chip stopped responding" / StopIteration no esptool
  (gravação para no meio — não é erro do sketch)
  → Feche o Monitor Serial e qualquer app que use a mesma COM.
  → Ferramentas → Velocidade de upload → 115200 (ou 460800; evite 921600 se instável).
  → Cabo USB com dados (não só carga); porta USB direta no PC, sem hub fraco.
  → Se precisar: segure BOOT, clique Upload, solte BOOT ao começar a gravar (algumas placas).
  → Alimentação: fonte 5 V estável no VIN se o USB oscilar; desligue periféricos que puxem muito 3,3 V.
  → Outro cabo/porta USB; reinicie o PC e o ESP32 (unplug 10 s).

• Linhas "note: #pragma message: BT: forcing BR/EDR..." no log
  → Aviso do SDK Bluetooth, pode ignorar (não impede compilar nem gravar).
